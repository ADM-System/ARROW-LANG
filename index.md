# Welcome

Arrow is a minimalist programming language built around one simple idea:

> **Data flows through the program.**

Instead of focusing on objects, Arrow focuses on transformations.

Its syntax is designed to be:

- Simple
- Readable
- Minimal
- Consistent

---

# Philosophy

Arrow follows a few simple rules.

- Data should flow.
- Functions should transform data.
- The language should remain small.
- The Standard Library should provide the power.
